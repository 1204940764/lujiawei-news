本人来自苏州工学院(二本)，本科背景不是很好，有过ACM经历，但最高奖项也只有邀请赛铜牌。与今年报华师的ap相比，实力只能算中下。我本科期间也非常的混，所有考试都是临时抱佛脚速刷通过的。接下来分享一下备考经验。

# 政治

我政治开始时间是9月份，一开始看的徐涛的课，看了马原、史纲、毛概，还买了一本苏一的思维导图带着看。对于选择题，我一开始刷的肖1000，后期的话大部分机构都会出模拟卷，自己可以在网上买一个微信小程序，每天刷几套模拟卷。然后我还买了一本澄潇宇的一页纸（plus版，大概12月会出，整合了所有大部分模拟卷的选择题知识点），当时学累了就看这本。考前建议也看一下澄潇宇的选择题技巧，然后刷几套真题找找感觉（例如多选题中选项与题干无关这种设置错误的方式）。上海政治主观题均分是27，也算小旱，而且主观题很难提分。我身边就有一个背主观题的人，背的比我多多了，最后主观题也和我差不多，所以对于政治大题我觉得背一下肖四就够了（我还没背完，背了60%左右），考试的时候尽量写满。

# 英语

英语这门课我一直都不好，考了73，在上岸的人里面不算高，就简单说一下吧。也讲一下我的情况，我英语不知道任何语法，完全就是0。	我的学习方法就是3月开始一直背单词（软件是不背单词），背到5月左右每天刷一篇阅读真题，把不会的单词添加到生词本中，然后每隔一段时间看一下生词本里的词。	作文我跟的是石雷鹏，最后考场写的也是模版（自我感觉写得很垃圾）。

# 数学

这门课我满分，完全就是火力够猛，刷了80套模拟卷，所以基本上大部分的题我都见过。

数学我从一月份就开始准备，跟的是武忠祥，一开始只是对着书学的，然后每学一点就去刷660的章节，寒假就把660结束了。开学时候我给了我朋友一些数学题，有些题他们一下就秒了，我却因为没见过都不会。所以我又花了一周不到的时间速通武忠祥基础课，把漏洞全给补掉了。然后又看了25年的强化课（看25年是因为26年的还没出），花了大概半个月，然后又把严选题给写了（很难的题没写），顺带着把张宇1000题基础部分的选择和填空也写了（注意！！！是基础部分，强化太难了我没写），至此高数强化结束。此时大概4月底。

后面就开了线性代数，基础和强化跟的姜小千，买了他的讲义。线代强化结束的时候发现写题还是有些困难，又买了一本李永乐的线性代数辅导讲义，就直接对着硬啃，这本书看完之后很多东西我都豁然开朗了（可以说是质变了，所以强烈建议买一本）。至此线代强化结束。此时大概5月底。

高数和线代强化结束后，我就买了880，花了大概一个多月的时间把基础和强化写了（选择、填空、大题全写）。在刷880期间，我又去看了 没咋了 的线性代数强化课（强烈建议，讲的很好），选择题技巧啥的也顺带一并看了。此时大概7月中旬。

之后就开始了我的漫长刷试卷旅程，先写的真题试卷，03年开始每天一套，一直刷到8月初，之后就开始模拟卷，期间还参加了一次欧几里得的模拟考，当时132分。我刷的模拟卷有：24合工大超越卷，25合工大超越卷，26合工大超越卷，24李林四套卷，25李林四套卷，26李林四套卷，24李林六套卷，25李林六套卷，26李林六套卷，26李艳芳三套卷（就写了两套，太难了），26欧几里得三套卷，25张宇四套卷，26张宇四套卷，26张宇八套卷。对于模拟卷，我建议大家都刷市面上比较火的，b站有人讲解。刷试卷期间我还看了武忠祥17堂课，epsilon-delta的大雪深埋课程。刷完这些大概考研前几天了。

数学这一门我花了大量的时间，后期刷模拟卷期间，我每天也会去看一些博主的知识点分享（例如小荷花啥的），反正没事就去刷视频课。

# 408计算机专业基础综合

408被誉为工科考研天花板，对于跨考而言看上去是4门课，其实是5门课，因为你还要学C语言，所以非科班谨慎选择！！！在考研三刷期间，我也做了思维导图，放在了github中，大家可以去下载使用。我建议大家也可以做属于自己的一份思维导图。

408我也是从一月份就开始了，我的学习顺序是数据结构（DS）->计算机组成原理（CO）->操作系统（OS）->计算机网络（CN）。所有课程跟的是王道，在三月初结束DS和CO，五月初结束OS和CN。一刷的时候每次都是刷一章视频然后跟着刷一章选择题。

五月初开始二刷王道视频，一直到七月初，二刷的时候我也重新刷了一遍选择。

七月初开始三刷王道视频，一直刷到九月初开学。在此期间我重新刷了一遍选择错题，并且把所有大题也写了（除了数据结构代码题有些没写），并且每看一章视频就做一章的思维导图（思维导图很重要！）。

九月开始刷真题，因为在王导课后题都写过，基本都是140+。九月底开始刷模拟题的选择（只刷选择），刷的试卷有：天勤，王道，零壹，图领，竟成，研芝士等等，对于模拟卷大家看自己，不一定非要刷。此时大概十一月，开始二刷真题+模拟题选择，还去看了beok的基础课和强化课，王道的强化课也看了一点。408我后期花的时间不太多，因为基本都学完了，每天也就是翻翻思维导图+看王道的书+看错题。就这样一直持续到考前。

对于408我前中期花的大量的时间都在视频上，后期时间基本都在写题和完善思维导图上。思维导图真的很重要，你可以把你不会的知识点放进去，然后经常翻就会全记住！

英语没啥好说的，正常发挥。。。

# 择校

关于择校。。。一定要根具自己的实力来选择，并且充分考虑到**复试**是否来得及准备以及是否有优势。我是到8～9月份那会才选择报考师大，以及自己有过ACM经历，较其他人而言有一定优势。前期（大概4-5月）我想冲的学校是南邮、华东理工、上海大学这些的，拿江苏大学、苏科大保底。所以大家在前期也不要太焦虑。

我前期不清楚自己的实力，觉得报个一本差不多了，往后学找到了自己的节奏，可以冲击更高的院校。。。

# 省钱技巧

考研期间我没有报任何补习班啥的，几乎也没买什么网课。整体花销大概不超过2000。

大部分网课及资料在百度网盘上都有，并且也够用了。https://www.kdocs.cn/l/cgQxEOyblJwA。这个链接里面有大部分的网课及资料，如果没有大家要的也可以去咸鱼买链接、二手书籍啥的。对于后期，大部分的模拟卷都可以找到pdf，并且在淘宝上打印（可能比较模糊），这样可以省不少钱。至于一些书，常用的可以买正版的，不经常用的（比如就打开几次就不看的）可以买便宜点的（懂得都懂）。经常用的书大家一定要买正版的，有些便宜的书纸的质量都不好，而且有些味道很重。
